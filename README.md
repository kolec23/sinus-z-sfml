W celu skompilowania programu należy:
-pobrać biblioteke sfml 2.4.0 GCC 4.9.2 TDM (SJLJ) ze strony: https://www.sfml-dev.org/download/sfml/2.4.0/ 
-rozpakować ją
-w opcjach projektu w zakładce Directories, scieżki Library Directories i Include Directories na ściezki z pobranego projektu
Potrzebne pliki dynamicznych są dołączone do projektu nie trzeba ich zmieniać.
-potym można  skompilować
-uruchomić z folderu lub zrobić skrót.

This program counts the equations of type sin(x)=m. It uses Dev-C++ 5.11 with compilator TDM-GCC 4.9.2 64-bit Release. 
If you want to compile, you will do:
-download the sfml 2.4.0 GCC 4.9.2 TDM (SJLJ) with website  https://www.sfml-dev.org/download/sfml/2.4.0/ 
-you will have to unpack zip with sfml lobrary
-in Project Options you will find Directories tab and  you will change Library Directories and Include Directories on paths with unpacked zip.
-Needed dynamic library files is included to project, so you haven't to change.
If you do this, you able to compile and run project. You may run in project folder or create shortcut.
